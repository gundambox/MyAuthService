# Code of Conduct

## Scope

This is a solo project maintained by gundambox. While this Code of Conduct outlines expected standards for any interactions, please note that this project does not actively seek external contributions and community interaction is limited.

## Standards

### Expected Behavior

- Use respectful and professional language
- Accept constructive criticism gracefully
- Focus on technical merit and project objectives
- Respect the maintainer's decisions regarding the project direction

### Unacceptable Behavior

- Harassment, discriminatory remarks, or personal attacks
- Trolling, insulting comments, or deliberately disruptive behavior
- Unsolicited promotion or spam
- Publishing others' private information without permission

## Reporting

For concerns regarding interactions in this project, please open an issue or contact the maintainer directly.

## Enforcement

As the sole maintainer, I reserve the right to:

- Remove, edit, or reject comments, commits, issues, and pull requests
- Block contributors who violate this Code of Conduct
- Make final decisions on all project-related matters
